# アネラカフェ ブランドガイドライン（想定）

> ※ 本部の正式なブランドガイドが提供され次第差し替える前提の「想定版」です。
> 「Anela（アネラ）」がハワイ語で **天使** を意味することを起点に設計しています。

## ブランドコンセプト
**「天使のようにやさしい、海辺のような癒しの居場所」**

- ハワイ（Anela＝天使）の、あたたかく穏やかな空気感
- 保護犬・就労支援を包み込む「やさしさ・癒し・ぬくもり」
- カフェとしての心地よさ（コーヒー・木のぬくもり・自然光）

キーワード：*やさしい / 癒し / あたたかい / 穏やか / 開かれた / 自然*

## カラーパレット

| 役割 | 名称 | HEX | 用途 |
|------|------|-----|------|
| ベース | Angel Ivory | `#FBF8F2` | 背景（光・天使の白） |
| サブ背景 | Soft Sand | `#F0E8D8` | セクション切り替え・カード背景 |
| プライマリ | Anela Teal | `#5C9EAD` | ブランド主色（ハワイの海・癒し） |
| プライマリ濃 | Deep Teal | `#3C6E79` | 見出し・フッター・ロゴ文字 |
| アクセント | Halo Gold | `#D9A65A` | 天使の輪＝光。ラベル・装飾の差し色 |
| CTA | Aloha Coral | `#E0876B` | ボタン・行動喚起（あたたかさ） |
| 自然 | Leaf Sage | `#8FA67E` | 犬・自然・里親まわりの差し色 |
| 文字 | Warm Ink | `#3D372F` | 本文 |
| 文字薄 | Soft Ink | `#6E6456` | 補足テキスト |
| 罫線 | Line | `#E7DDCB` | 枠線・区切り |

配色比率の目安：**ベース(Ivory) 70% / Teal 15% / Gold・Coral・Sage 合わせて 15%**。
差し色を使いすぎず、余白とアイボリーで“やさしさ”を保つ。

## タイポグラフィ
- **見出し**：明朝系（Hiragino Mincho / Noto Serif JP）— 上品で穏やかな印象
- **本文**：ゴシック系（Hiragino Kaku / Noto Sans JP）— 読みやすさ重視
- 行間ゆったり（本文 line-height 1.8）、余白広め

## トーン & マナー
- 角丸でやわらかく（radius 14〜22px）、影は薄く自然に
- 写真は自然光・暖色寄り。犬・人・コーヒーの“ぬくもり”が伝わるもの
- 文章は丁寧でやさしい敬体。専門用語は噛みくだく（特に就労支援・里親）

## ロゴ
- 現状リポジトリにあるのは「キノコキャンプ」ロゴのみ。**アネラカフェのロゴは要支給**。
- 暫定でテキストロゴ（明朝「アネラカフェ」＋店名）を使用。

## 実装
このガイドラインは `packages/shared/styles/global.css` の `:root` カスタムプロパティに反映済み。
色を変えたいときは原則 `global.css` の変数を1か所だけ編集すれば全ページ・両サイトに反映される。
