# 画像の置き場所（原田店）

ここに写真を置いて、`src/data/store.ts` でファイル名を指定すると表示されます。

- ヒーロー画像: `hero.jpg` を置き、`store.ts` で `heroImage: "/images/hero.jpg"`
- ギャラリー: `g1.jpg` … を置き、`store.ts` の `gallery` に
  `{ src: "/images/g1.jpg", alt: "説明" }` を追加

原田店の Instagram アカウントが決まったら `store.ts` の `instagram` も差し替えてください。

推奨: 正方形（1080×1080程度）/ JPEG / 1枚あたり数百KBにリサイズ。
